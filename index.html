Prisma Game Lab

Na palestra do Prisma Game Lab, fundamentos para a criação de jogos foram esclarecidos e explicados. Para criar um jogo, as funções são divididas entre alguns grupos, entretanto não necessariamente precisa de várias pessoas para fazer um Game, isso depende do quão complexo será o produto final.

No processo de criação podemos assinalar:

  • Game Design -> Idealizam a parte criativa do jogo
  • Artistas -> Participam da ambientação, criação dos personagens
  • Sonorização -> Efeitos especiais e trilha sonora
  • Programação -> Botam tudo para funcionar
  • Mídia Social -> ‘‘Propaganda’’, divulgação

Para a criação de um personagem, diversos elementos devem ser levados em conta, portanto fazer uma ficha técnica é o primeiro passo. Para a ficha técnica devemos levar em consideração algumas perguntas para facilitar o trabalho: Como é seu corpo? Que história? Como é seu mundo? Que sentimento ele carrega? O que ele faz? Quais as mecânicas?

São nesses pequenos detalhes que tornam um personagem memorável, como é o caso de alguns personagens de jogos famosos como o Sonic, Mario Bros, Hollow Knight.

A construção visual do personagem é outro processo para a criação, formas geométricas, linha de ação, cores, proporção devem estar participando intrinsecamente para o processo de formação do personagem.

Exercício proposto em aula

Foi pedido que a realização de forma rápida a criação de um personagem por meio de 3 elementos apresentados: Machado - Leão - Fogo

Por meio dos elementos tentei criar um personagem bárbaro em que tivesse todos esses elementos, entretanto não achei que realizei o mesmo da melhor forma.
